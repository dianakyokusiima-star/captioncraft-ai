# CaptionCraft AI

Transform your ideas into scroll-stopping captions in seconds.

## Overview

CaptionCraft AI is an AI-powered caption generation tool designed for creators, entrepreneurs, marketers, influencers, and small business owners who want engaging social media captions without spending hours brainstorming.

Whether you're posting on Instagram, Facebook, LinkedIn, TikTok, Pinterest, or X, CaptionCraft AI helps you create compelling content that attracts attention, encourages engagement, and saves time.

## Installation

```bash
git clone https://github.com/yourusername/captioncraft-ai.git

cd captioncraft-ai

npm install

npm run dev
```

Open:

```bash
http://localhost:3000
```

## Environment Variables

Create a `.env.local` file:

```env
OPENAI_API_KEY=your_api_key

DATABASE_URL=your_database_url

NEXTAUTH_SECRET=your_secret_key
```
